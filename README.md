# Smart Farming Website (JavaScript Based)

Smart Farming is a JavaScript-based web application designed to support farmers with a simple, user-friendly digital platform. The project demonstrates practical implementation of HTML, CSS, and JavaScript for building responsive and interactive web interfaces.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Features
- Responsive design
- Clean and minimal UI
- Interactive functionality using JavaScript
- Optimized and structured code

## How to Download & Run

1. Click **Code** → **Download ZIP**
2. Extract the ZIP file
3. Open the project folder
4. Double-click `index.html`

The website will run directly in your browser (No installation required).
